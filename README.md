# SWP (Sovereign Workflow Protocol) 프로젝트

## 🏴‍☠️ 프로젝트 개요
하꼬방 해적단의 자율형 AI 운영 플랫폼

## 📅 업로드 정보
- 업로드 날짜: 2026-03-09
- 프로젝트 버전: v1.0
- 포함 내용: 캡틴스 대시보드, 나노봇 실험실, Sovereign Memory, ZeroMQ 통신

## 🚀 주요 기능
1. 통합 캡틴스 대시보드 (Streamlit)
2. 나노봇 실험실 (자동 스킬 평가 시스템)
3. Sovereign Memory Fortress (로컬 벡터 DB)
4. ZeroMQ 기반 실시간 통신

## 📁 파일 구조
- `dashboard/`: 통합 대시보드 관련 파일
- `nanobot_lab/`: 나노봇 실험실 파일
- `sovereign_memory/`: 기억 요새 파일
- `zmq_communication/`: ZeroMQ 통신 파일
- `docs/`: 문서 파일

## 🔧 기술 스택
- Python 3.11+, Streamlit, FastAPI
- SQLite, SQLite-Vector
- ZeroMQ (pyzmq), Docker
- Ollama (로컬 LLM)
