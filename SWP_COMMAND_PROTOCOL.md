# Command Protocol for Depidi → Nanobot Communication
# 작성: 2026-03-04 13:19

---

## 프로토콜 개요

**목적**: 데피디 ↔ 나노봇 간 비동기 명령 전달 체계 확립

**구조**:
1. 데피디가 명령서 작성 → `/home/hakkocap/다운로드/swp/commands/`
2. 캡틴이 나노봇에게 "SWP TASK 확인하라" 전달
3. 나노봇이 명령서 읽고 실행
4. 나노봇이 결과를 보고서로 작성 → `/home/hakkocap/다운로드/swp/reports/`
5. 데피디가 보고서 확인

---

## 명령서 표준 형식

### 파일명 규칙
```
COMMAND_[YYYY-MM-DD]_[HHMM]_[제목].md
```

예시:
```
COMMAND_2026-03-04_1319_SkillCollection.md
```

### 내용 구조
```markdown
# SWP 명령서
**발신**: 데피디 부관  
**수신**: 나노봇 #0  
**시각**: YYYY-MM-DD HH:MM  
**임무 코드**: [짧은 코드]  
**우선순위**: P0 (긴급) / P1 (높음) / P2 (보통) / P3 (낮음)

---

## 임무 목표
[명확한 한 문장]

## 실행 단계
1. [구체적 명령 1]
2. [구체적 명령 2]
3. [구체적 명령 3]

## 출력 형식
[JSON/Markdown/텍스트 등 명시]

## 저장 경로
[결과 파일 위치]

## 완료 조건
- [ ] 조건 1
- [ ] 조건 2
- [ ] 조건 3

---

**이 명령서를 읽었으면 즉시 실행하고 보고서를 작성하라.**
```

### 파일명 규칙
```
REPORT_[YYYY-MM-DD]_[HHMM]_[임무코드].md

```

- [파일 경로 1] (크기)

## 캡틴 지시 표준 문구

캡틴께서 나노봇에게 이렇게만 말씀하시면 됩니다:

```
나노봇, SWP TASK 확인하라
```

또는

```
나노봇, SWP 명령 실행
```

나노봇은 이 말을 들으면:
1. `/home/hakkocap/다운로드/swp/commands/` 폴더 확인
2. 가장 최근 `COMMAND_*.md` 읽기
3. 실행
4. `REPORT_*.md` 작성

---

## 긴급 명령 (P0)

긴급한 경우 캡틴이:
```
나노봇, SWP 긴급 명령 실행
```

나노봇은 P0 우선순위 명령만 찾아서 즉시 실행.
