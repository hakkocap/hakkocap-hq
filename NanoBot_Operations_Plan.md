## 🎯 임무별 역할 분담

### 나노봇 1호: 정찰병 (Scout)
- **역할:** 외부 정보 수집
- **주요 기능:**
  - RSS 피드 파싱 (Hugging Face, TechCrunch, arXiv)
  - 웹 검색 (Brave Search - 무료 한도 내)
  - 데이터셋 스캔
- **도구:** ai_trend_collector.py
- **연료:** 무료 (로컬)

### 나노봇 2호: 정제병 (Refiner)
- **역할:** 원시 데이터 가공
- **주요 기능:**
  - HTML 파싱 (Scrapling Adaptor)
  - 메타데이터 추출
  - JSONL 변환
- **도구:** dataset_plunderer_v2.py
- **연료:** 무료 (로컬)

### 나노봇 3호: 분석병 (Analyst)
- **역할:** 데이터 심층 분석
- **주요 기능:**
  - 텍스트 요약
  - 키워드 추출
  - 트렌드 분석
- **도구:** gemma3-pirate (로컬)
- **연료:** 무료 (로컬)

### 나노봇 4호: 품질관리병 (Quality Controller)
- **역할:** 출력 품질 검증
- **주요 기능:**
  - Fact-Check (Fact_Check_System.py)
  - 신뢰도 점수 산출
  - 적색 플래그 감지
- **도구:** gemma3-pirate (로컬)
- **연료:** 무료 (로컬)

### 나노봇 5호: 전략가 (Strategist)
- **역할:** 최종 의사결정
- **주요 기능:**
  - 복잡한 코드 분석
  - 고도 전략 수립
  - 캡틴에게 보고서 작성
- **도구:** MiniMax-M2.5 (API)
- **연료:** $0.05/일 제한
