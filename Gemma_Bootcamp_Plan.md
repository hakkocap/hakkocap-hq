## 🎓 3개월 커리큘럼

### 🔹 1개월차: 기초 이식 (Foundation Transfer)

**목표:** Gemma-3의 기본 동작 이해 + 나노봇 조작 역량 확보

| 주 | 교육 내용 | 실습 과제 |
|----|----------|----------|
| 1주차 | QLoRA/LoRA 이론 학습 | PEFT 라이브러리 설치 및 기본 설정 |
| 2주차 | Hugging Face TRL 사용법 | TRL로 간단한 파인튜닝 스크립트 작성 |
| 3주차 | 데이터 전처리 파이프라인 | 나노봇 데이터 →训练용 JSONL 변환 |
| 4주차 | **1차 실전演习:** 캡틴 철학 데이터로 Fine-tuning | gemma3-pirate에 Captain_Philosophy.txt 학습 |

**성공 기준:** 로컬에서 1GB 이하 데이터셋 파인튜닝 완료

---

### 🔹 2개월차: 전술 고도화 (Tactical Advanced)

**목표:** 전문 분야 특화 + 다중 모델协作

| 주 | 교육 내용 | 실습 과제 |
|----|----------|----------|
| 5주차 | 4-bit 양자화 (QLoRA) 적용 | VRAM 사용량 50% 절약 검증 |
| 6주차 |Function Calling / Tool Use | 외부 API 연동 스크립트 개발 |
| 7주차 | 다중 모델 앙상블 | gemma3 + MiniMax协作 체인 구성 |
| 8주차 | **2차 실전演习:** tech_essence 데이터로 전문화 | 기술 문서 분석 전용 모델 탄생 |

**성공 기준:** 특정 도메인에서 응답 품질 20% 향상 입증

---

### 🔹 3개월차: 자율 판단 (Autonomous Judgment)

**목표:** 인간 개입 없이 전략적 의사결정 가능

| 주 | 교육 내용 | 실습 과제 |
|----|----------|----------|
| 9주차 | RAG (Retrieval-Augmented Generation) 구축 | 나노봇 데이터베이스 연동 |
| 10주차 | Self-Correction 로직 구현 | 오류 발생 시 자동 복구 |
| 11주차 | 비용 최적화 자동화 | API 사용량 /전기세 모니터링 대시보드 |
| 12주차 | **최종 작전:**全 나노봇 통합 작전 | 해적단 전 체계 자동화 완성 |

**성공 기준:** 24시간 무인 가동 가능
